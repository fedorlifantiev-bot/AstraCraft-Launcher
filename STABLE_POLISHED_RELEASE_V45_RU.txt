AstraCraft Launcher v45 — стабильный полированный релиз

Что сделано:
- база: интерфейс v27/v43 без новостей, X-Ray и hitboxes;
- оставлен красивый Smart Center вместо новостной плашки;
- Java запускается через java.exe, чтобы ошибки писались в лог, а не только в системное окно;
- добавлен безопасный лимит RAM, чтобы не ловить Could not create the Java Virtual Machine;
- JVM-аргументы очищаются от опасных/дублирующихся -Xmx/-Xms;
- если Minecraft закрывается сразу, лаунчер пишет понятную ошибку и лог в .minecraft/logs/astracraft-launch.log;
- язык лаунчера сохраняется и дополнительно записывается в .minecraft/options.txt через lang:...;
- убраны карточки сидов/здоровья/брони с главной, чтобы публичная версия выглядела чисто;
- README и версия проекта обновлены до 4.0.0.

Рекомендуемый запуск:
1. run_dev_windows.cmd
2. В настройках включить Auto-download dependencies.
3. Нажать Smart RAM.
4. Нажать Verify & Repair для выбранной версии.
5. Запустить игру.
