AstraCraft Launcher — старт

1. Распакуй архив в отдельную папку без кириллицы и спецсимволов в пути.
2. Для запуска из исходников: run_dev_windows.cmd
3. Для сборки exe: publish_windows_single_exe.cmd
4. Если Minecraft не стартует:
   - нажми Smart RAM;
   - поставь Java auto / включи Auto-download dependencies;
   - нажми Verify & Repair;
   - проверь .minecraft/logs/astracraft-launch.log.

В этой версии нет X-Ray, hitboxes automation, noclip и loading-screen модов.
