# Release Checklist

Before publishing a public release:

- [ ] README is updated.
- [ ] LICENSE is present.
- [ ] SECURITY.md and PRIVACY.md are present.
- [ ] `.gitignore` excludes `bin/`, `obj/`, `.minecraft/`, Java runtimes and installer outputs.
- [ ] Build from a clean checkout works.
- [ ] Release tag exists, for example `v1.0.0`.
- [ ] EXE/MSI/MSIX files are attached to GitHub Releases, not committed to source.
- [ ] SHA256 hashes are published.
- [ ] Antivirus notice is honest and does not claim impossible guarantees.
- [ ] Support bundle was tested.
